# 🌿 VeePortfolio

A modern and responsive personal portfolio website built to showcase my projects, technical skills, education, certifications, and experience as a **Computer Engineering Undergraduate**, **Cyber Security Enthusiast**, and **Full Stack Developer**.

## 🚀 Live Demo

> [https://(https://veeportfolio-varnaja.vercel.app/)



---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern glassmorphism UI with botanical-inspired theme
* Animated hero section with typing effect
* Featured projects showcase
* Education and certifications slider
* Professional experience timeline
* Skills and technology stack sections
* Contact form powered by EmailJS
* Smooth scrolling and interactive animations
* Mint-green custom design theme

---

## 🛠️ Technologies Used

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* React Icons
* EmailJS
* JavaScript (ES6+)

## ⚙️ Installation

Clone the repository:

git clone https://github.com/VeeQubit/my-portfolio.git


Navigate to the project:

cd my-portfolio


Install dependencies:

npm install


Start the development server:

npm run dev


Build for production:

npm run build
